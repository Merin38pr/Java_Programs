# Student Management System (Java)

A console-based Java application to manage student records with file storage support. The system allows users to perform basic CRUD operations and stores data permanently using a text file.

## Features

- Add new student
- View all students
- Update student details
- Delete student
- Data persistence using file storage

## Technologies Used

- Java
- File Handling (BufferedReader, BufferedWriter)
- OOP Concepts
- ArrayList

## How It Works

- Student data is stored in a file named `students.txt`
- Data is loaded automatically when the program starts
- Any changes (add, update, delete) are saved to the file
